# Однодневный планировщик задач

Пример одностраничного приложения на React для управления задачами в пределах одного дня. Поддерживает проекты, задачи и подзадачи, отметку статуса и Pomodoro‑таймер. Все тексты интерфейса на русском языке.

Откройте `index.html` в современном браузере. Для разработки не требуется установка зависимостей.
